Raven Widget - Production

This page demonstrates the Raven chat widget using the production API.

Configuration:
API URL: https://raven-production-980b.up.railway.app
Business ID: YOUR_BUSINESS_ID

How to embed on your site:

  1. Get your Business ID from the Raven dashboard
  2. Copy the script code below
  3. Paste it before the closing </body> tag on your website
<!-- Raven Chat Widget -->
<script>
  window.RAVEN_CONFIG = {
    businessId: 'YOUR_BUSINESS_ID',
    apiUrl: 'https://raven-production-980b.up.railway.app'
  };
</script>
<script src="https://raven-production-980b.up.railway.app/static/raven-widget.js"></script>
